Prework Study Guide

HTML

  • The head element contains information about the webpage
  • The body element represents the visible content shown to the user
  • The title element is important for SEO and driving traffic to your website
  • The meta element is for information that is only used by the web browser
  • The a element is used to create links that direct the user to the same webpage or other webpages

CSS

  • A margin indicates how much space we want around the outside of an element
  • A padding indicates how much space we want around the content inside an element
  • text-align specifies the alignment of text along the x-axis
  • The selector defines the element or attirbute to which declarations will apply
  • Declarations contain the property being applied and the value of the property
  • We must use the link tag to attach a style sheet to an HTML webpage

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and swithces to it
  • git add: adds modifications in the current branch to the staging area
  • git commit: captures a snapshot of the currently staged changes
  • git pull: used to update the local repo to match the content of a remote repo
  • git push: pushes changes from our local repo to a remote GitHub branch
